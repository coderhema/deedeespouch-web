# Deedees Cooking Pouch Web

Premium Next.js + Tailwind storefront for @deedeespouch.

## Brand audit summary
- Brand identity: traditional Nigerian food, especially Moi Moi, Agidi, and Ekuru.
- Positioning: premium, convenient, family-friendly, and diaspora-oriented.
- Visual direction: purple and white, editorial spacing, clean typography, restrained motion.
- Messaging: cook smart, eat better, tradition in every pouch, Australia delivery.

## Strategy
- Lead with appetite and brand story.
- Reinforce trust with delivery clarity and simple product language.
- Use a premium component system to keep the experience calm and polished.
- Keep the page conversion-focused with one clear CTA path.
